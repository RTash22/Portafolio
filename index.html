RoboKids - Juego Educativo de Robótica

Juego educativo interactivo desarrollado en React que enseña conceptos básicos de programación y lógica a niños a través de un robot que debe resolver problemas matemáticos. Características principales: modo profesor para crear niveles personalizados, modo 1vs1 para competencia entre estudiantes, sistema de rondas configurables, animaciones fluidas y retroalimentación visual.

React JavaScript CSS3 Educación

DentistaApp

Aplicación móvil para gestión de citas dentales desarrollada con React Native. Incluye sistema de agenda, gestión de pacientes, historiales médicos, servicios y procedimientos. Características principales: gestión de citas en tiempo real, perfiles de doctores y pacientes, historial médico detallado, y sistema de notificaciones.

React Native Expo JavaScript APIs REST
Repository

API Consultorio Dental

Backend desarrollado en Laravel para la aplicación DentistaApp. Proporciona una API REST completa para la gestión de citas dentales, pacientes, doctores, servicios y procedimientos. Incluye autenticación, manejo de estados de citas, historial médico, y sistema de notificaciones por correo. Desplegado en Heroku con base de datos PostgreSQL.

Laravel PHP PostgreSQL REST API
